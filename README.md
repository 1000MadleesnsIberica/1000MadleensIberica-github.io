# 1000MadleensIberica-github.io
1000MadleensIbérica, que abarca la Península Ibérica, se encuentra bajo el paraguas de Mil Madeleens a Gaza, una campaña mundial de flotillas marítimas para romper el bloqueo ilegal de Gaza mediante la solidaridad no violenta, inspirada en la firmeza, impulsada por la esperanza y unida en la acción.
